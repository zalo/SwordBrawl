# SwordBrawl

A browser-based sword fighting demo powered by physics simulation (PhysX WASM) and neural network policies (ONNX Runtime).

## Play

Visit the live demo: https://selstad.github.io/SwordBrawl/

## Controls

- **WASD** — Move
- **Right-click drag** — Rotate camera
- **Scroll** — Zoom
- **Left click** — Slash
- **Space** — Jump
- **Q** — Kick
- **E** — Block

Touch controls (joystick + action buttons) are available on mobile devices.

## Tech

- PhysX WASM for rigid body simulation
- ONNX Runtime Web for neural network inference
- Three.js for rendering
- Hierarchical reinforcement learning (HRL) for character control
